Liver Patient Prediction Model

Project Date April 2024
Role ML Developer
Client Personal Project
Visit GitHub Repo

This project focuses on early detection of liver disease using machine learning. A classification model was trained on patient medical records to distinguish between individuals with and without liver conditions.

The dataset, sourced from publicly available medical repositories, includes features such as bilirubin levels, enzyme concentrations, age, and albumin levels. These clinical indicators were used to train and evaluate multiple algorithms.

We applied preprocessing techniques to clean the data, handle missing values, and normalize feature values for better model accuracy. Feature selection and correlation analysis were visualized using Seaborn heatmaps to eliminate redundancy.

We tested multiple classifiers such as Logistic Regression, Random Forest, and Decision Trees. The final model was evaluated using metrics like accuracy, F1-score, precision, and recall. Visualization of model performance was done through confusion matrices and ROC curves.
