Spotify Music Recommendation System

Project Date April 2024
Role ML Engineer
Client Academic Project
Visit GitHub Repo

This system suggests songs based on user preferences using collaborative filtering techniques. Built using Python, Pandas, NumPy, SciPy, and Scikit-learn, with a user-friendly interface via Streamlit. Data was preprocessed from user-listening behavior datasets.

Collaborative filtering was used to generate personalized music recommendations based on the listening history of similar users. The system enables users to discover new songs based on patterns of users with similar tastes.

The final model was deployed using Streamlit, enabling easy access to the recommender through a web-based interface, offering a seamless user experience.
